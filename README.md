### 1.图书管理系统的用例关系图  
##### 1.1用例图PlantUML源码如下：  
```usecase
@startuml
left to right direction
:图书管理员: as Admin
:读者: as User
Admin -->(借出图书)
Admin -->(归还图书)
Admin -->(维护书目)
Admin -->(维护读者信息)

User -->(查询借阅情况)
User -->(查询书目)
User -->(预定图书)
User -->(取消预定)
(预定图书) .>(查询书目):include
(维护书目) .>(查询书目):include
(取消预定) .>(预定图书):extend
(归还图书) .>(借出图书):extend
@enduml
```

##### 2.1图书管理员：  
主要职责是：借出图书、归还图书、维护图书和维护读者信息  
##### 2.2读者：  
主要职责是：查询图书、预定图书、查询借阅情况和取消预定
